c)说明模型训练过程：人工标注50+张图像 → 提取特征 → 训练SVM → 保存模型；
d)给出整体流程图或伪代码。
4.系统实现
a)说明工具开发环境（Python + Tkinter）；
b)展示主界面：左侧图像列表、中部方法选择、右侧结果展示；

5.实验与分析
a)构建测试集（30+张，含合格/不合格样本）；
b)对比实验：方法四（ML） vs 方法一至三（单指标）；

1.2题目8：基于多尺度边缘融合的钻孔图像圆心鲁棒检测方法
1.2.1负责人：张皓翔
1.2.2需求概述
本课题针对野外钻孔光学图像中圆心定位困难的问题（如因井壁泥浆遮挡、探头偏心、光照不均导致传统方法失效），研究一种基于多尺度边缘融合的圆心检测方法。系统需基于 Python 的 Tkinter 图形界面库开发一个桌面工具，支持用户加载图像、执行多种圆心检测算法、可视化检测结果（含圆心位置与拟合圆）、对比不同方法的鲁棒性，并输出量化评估指标，为后续图像展开或质量筛查提供可靠几何基准。
1.2.3详细需求
1.主窗口与基本布局
a)程序启动后显示一个主窗口，使用 Tkinter 构建图形界面，布局分为：
b)左侧：原始图像显示区 + “加载图像”按钮
c)中部：检测方法选择与控制面板
d)右侧：检测结果可视化区
e)底部状态栏：显示检测结果坐标、半径及评估信息
2.加载钻孔图像
a)用户点击“加载图像”按钮，从本地选择一张 JPG 或 PNG 格式的钻孔环形图像；
b)图像成功加载后，在左侧区域显示，并在状态栏提示“图像已加载”。
3.实现并集成四种圆心检测方法
系统需同时实现以下四种检测策略，供用户选择对比：
a)方法一：霍夫圆变换（Hough Circle）
i.使用 OpenCV 标准霍夫圆检测，采用默认参数组合；
b)方法二：轮廓最小外接圆（Min Enclosing Circle）
i.先提取图像最大闭合轮廓，再计算其最小外接圆；
c)方法三：基于Canny边缘的单尺度霍夫检测
i.先用固定阈值进行 Canny 边缘检测，再对边缘图执行霍夫变换；
d)方法四：多尺度边缘融合检测（本课题核心）
i.在多个 Canny 阈值下提取边缘，融合各尺度边缘响应，再进行霍夫投票，提升对弱边缘、断裂边缘的容忍度。
4.执行检测与结果可视化
a)用户在中部面板选择一种方法，点击“执行检测”按钮；
b)系统在右侧区域显示叠加了检测结果的原始图像：
i.以绿色十字标记检测出的圆心；
ii.以红色圆圈绘制拟合的圆轮廓；
c)若检测失败（如未返回有效圆），则弹出提示：“当前方法未能检测到有效圆心”。
5.多方法对比模式
a)提供“启用对比模式”复选框；
b)勾选后，右侧区域分为四格，同时显示四种方法的检测结果；
c)每格下方标注方法名称及检测到的圆心坐标（x, y）和半径 r。
6.鲁棒性评估与指标输出
a)每次检测后，系统自动计算并在状态栏显示以下指标：
i.边缘覆盖度：拟合圆周上被原始边缘覆盖的比例（越高越可靠）；
ii.圆心置信度（仅对霍夫类方法）：霍夫投票峰值强度；
b)在对比模式下，四项结果以表格形式并列展示，便于横向比较鲁棒性。
7.结果保存与导出
a)用户可点击“保存当前结果图”，将右侧带标注的图像保存为 PNG 文件；
b)在对比模式下，可点击“保存对比图”，生成一张四宫格结果图用于报告。
8.用户交互与容错处理
a)若未加载图像即点击“执行检测”，弹出 Tkinter 提示框：“请先加载钻孔图像”；
b)所有方法均需处理异常（如空轮廓、无效参数），避免程序崩溃
